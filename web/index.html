PostCrossing Mock

Minimal UI for requesting addresses and registering received postcards.
Default server from this project listens on http://localhost:4000.
Use one of the seeded IDs (e.g., alice, bruno, cami, dave).
💡 Tip: If you open this file directly from your file system, your browser may block requests due to CORS. Serve it via a local static server or host it under the same origin as the API to avoid CORS errors.

📮 Send Flow

ℹ️ After "Request Address", you'll get a postcard code like US-1 and a recipient snapshot. Use that code in the "Register Received" section below to simulate reciprocity.

📬 Register Received

ℹ️ When a postcard is registered as received, the sender gains 1 receive credit. The next "Request Address" will draw from the pool FIFO.
