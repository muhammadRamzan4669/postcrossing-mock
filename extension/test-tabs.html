PostCrossing Extension Test Page

Use this page to test the PostCrossing Tab Sorter extension

How to test:

  1. Right-click on the links below and select "Open link in new tab" for multiple tabs
  2. Open the links in random order to shuffle the tabs
  3. Click the PostCrossing Tab Sorter extension button in your toolbar
  4. Or use the keyboard shortcut: Alt + Shift + S
  5. Watch the tabs get sorted by their postcard numbers!
